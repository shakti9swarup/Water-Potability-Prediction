 ## Water💧 Potability Prediction using Machine Learning

## 📌 Project Overview
This project aims to predict whether water is **safe for drinking (potable)** or **unsafe (non-potable)** using Machine Learning techniques.  
It analyzes multiple **water quality parameters** and builds a classification model to determine water safety.

---

## 🎯 Objectives
- ✔️ Analyze water quality features  
- ✔️ Perform data preprocessing and cleaning  
- ✔️ Build and train machine learning models  
- ✔️ Evaluate model performance  

---

## 📂 Dataset Details

The dataset contains the following important **water quality parameters**:

| Feature Name        | Description |
|--------------------|------------|
| 💠 pH              | Acidity/alkalinity level |
| 💠 Hardness        | Mineral content in water |
| 💠 Solids          | Dissolved solids |
| 💠 Chloramines     | Disinfectant level |
| 💠 Sulfate         | Sulfate concentration |
| 💠 Conductivity    | Electrical conductivity |
| 💠 Organic Carbon  | Organic matter content |
| 💠 Trihalomethanes | By-products of chlorination |
| 💠 Turbidity       | Water clarity |

### 🎯 Target Variable:
- **0 → Not Safe 🚫**  
- **1 → Safe ✅**
